# Harbor

## rust supply chain security github integration, via cargo-vet

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on http://localhost:3000

```bash
yarn dev
```

## Environment Variables

```
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
REDIS_LINK=
REDIS_PASSWORD=
```
